README

This is the data generated from the Java simulations for up to 150 packets, using increments of 5 with 50 trials per increment. The cost of a collision was log_2(n) --- note this is the same for our 1 million packet trials --- for whatever n was in the current trial (5, 10, 15,..., 150).
